# Power BI Business Dashboard

## Project Overview

This project demonstrates the creation of an interactive business dashboard using Power BI to analyze business performance and visualize key metrics.

## Objectives

- Monitor business KPIs
- Analyze sales performance
- Compare revenue trends
- Build interactive visualizations

- Excel Dataset

- Business Analytics
